Usage Instructions

Getting Started
  1. Import the admin-modern-box.css file in your admin programs CSS
  2. Use the .admin-program-box class as the main container
  3. Add the .admin-program-box-content for inner content
  4. Use appropriate status indicators (.admin-draft-indicator, etc.)
  5. Customize colors and spacing using CSS variables
CSS Variables Required

Make sure your project defines the required CSS variables for colors, spacing, and typography. These should be available in your base.css or shared styles.

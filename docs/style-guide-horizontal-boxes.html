Horizontal Box Layout with Tabs

Design Benefits

  • Rich Information Display: Program descriptions provide context without cluttering the interface
  • Editor Visibility: User avatars show who can edit each program, with "All" for agency-wide access
  • Project Timeline: Positioned in same column as actions but at metadata row level with vertical spacing
  • Clickable Program Names: Direct navigation without needing separate view buttons
  • Compact Initiative Display: Icon + number with tooltips saves horizontal space
  • Clear Status Indicators: Simple colored circles with descriptive text
  • Streamlined Actions: Single dropdown button with contextual menu items
  • Mobile Responsive: All elements adapt gracefully to smaller screens

Implementation Notes

  • Program Descriptions: Brief description from programs table displayed below program name
  • Editor Avatars: 24px circular avatars with hover tooltips, "All" text when all agency users are editors
  • Timeline Display: Single calendar icon with date range, positioned at bottom-right of program box
  • Initiative Tooltips: Hover over initiative icons to see full initiative names
  • Status System: Colored circles (Active=green, Completed=blue, Pending=yellow, Inactive=gray)
  • Simplified Timeline: No start/end labels, just calendar icon and date range with arrow
  • Avatar Integration: Uses same styling as navbar avatars for consistency
  • Conditional Rendering: Timeline section only appears when timeline data exists
